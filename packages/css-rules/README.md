# css-rules

> Returns a parse tree for a CSS source.

## Install

Install with [npm](https://npmjs.org/package/css-rules)

```
npm install --save css-rules
```

## Usage

```js
var parseCSS = require('css-rules');

var rules = parseCSS(css);
```
